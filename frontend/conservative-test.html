Conservative / Path-Independence Test

A CMF is conservative (or flat) if matrix products are path-independent: any two admissible paths between the same endpoints yield the same product. Tested empirically at a grid of lattice points.

Mathematical Background

A 2D CMF has matrix fields \(K_1(k,m)\) and \(K_2(k,m)\). The flatness condition states:

\[K_1(k,m) \cdot K_2(k{+}1,\,m) = K_2(k,m) \cdot K_1(k,\,m{+}1)\]

When this holds for all \((k,m)\), products along any lattice path from \((0,0)\) to \((K,M)\) agree.

This test checks the condition numerically at each grid point using 20-digit mpmath arithmetic.

The residual = Frobenius norm of \(K_1 K_2 - K_2 K_1\).

Note: Small residuals are empirical evidence only — not a formal proof. Symbolic verification via SymPy/Maple is required for certainty.

Method: At each \((k,m)\), evaluates \(\text{LHS} = K_1(k,m)\cdot K_2(k{+}1,m)\) and \(\text{RHS} = K_2(k,m)\cdot K_1(k,m{+}1)\) using 20-digit mpmath arithmetic. Residual = \(\|LHS - RHS\|_F\). For flat CMFs, residuals should be \(\approx 10^{-20}\).
